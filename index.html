시스템 아키텍처

🧠 Brain — Jetson Orin Nano
  • ROS2 Navigation Stack (SLAM + Nav2)
  • BME AI-Studio 냄새 라벨링 모델
  • 상황 판단 엔진 (위치 + 공기질 → 향 선택)
  • NVIDIA Riva (음성 인터페이스)
👃 Sensors — ESP32
  • BME688: 가스 핑거프린트 + 온습도기압
  • SGP41: VOC + NOx Index
  • RPLIDAR A1/A2: SLAM + 장애물 회피
  • USB/CSI 카메라: 상황인지
⚙️ Actuators
  • 공기청정기 (팬 + HEPA, 3D프린팅 제작)
  • 디퓨저 모듈 (서보 × 3~5, 향 선택 밸브)
  • 마이크 + LCD/OLED 패널
🚗 Mobility
  • DC 기어드 모터 × 2~4
  • 리튬 배터리 팩 (11.1V)
  • WiFi 5 + Bluetooth 5.0

주요 시나리오

💨 시나리오 A — 악취 감지 및 정화
  1. SGP41 VOC 급상승 감지
  2. Jetson: 냄새 추적 모드 ON
  3. LiDAR로 악취 방향 이동
  4. BME688 → "생선 탄내" 라벨 식별
  5. 공기청정기 최대 가동
  6. 스마트폰 알림 전송
🌸 시나리오 B — 상황 맞춤 디퓨저 분사
  1. 위치 + 공기질 + 시간대 종합 판단
  2. "숲속 향기" 선택
  3. "향기 분사할까요?" 음성 질문
  4. 마이크로 "응" 수신 확인
  5. 디퓨저 분사 시작
📱
📱 시나리오 C — 원격 제어
  1. 귀가 전 앱에서 "라벤더 향 분사" 명령
  2. WiFi로 Jetson에 전달
  3. 집 도착 전 미리 향 세팅 완료 🎉

개발 일정

Phase 0 (~2주)

설계 확정, 부품 발주, ROS2 환경 세팅, ESP32↔Jetson 통신 프로토타입

Phase 1 (2~4주)

SLAM 기본 동작, BME688+SGP41 수집, BME AI-Studio 냄새 학습, 디퓨저 HW 제작

Phase 2 (4~6주)

SLAM+센서퓨전 연동, 상황 판단 엔진, 음성 인터페이스, 공기청정기 3D프린팅 통합

Phase 3 (6~8주)

원격 제어(WiFi/BT), 알림 기능, 통합 테스트, 발표 데모 완성
